Guía Rápida de CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) permite aplicar estilos visuales a elementos HTML, como colores, tamaños y distribuciones. Es la herramienta que hace que tu web pase de ser un esqueleto a una obra de arte visual.

1. Selector

Indica a qué elementos HTML se aplicarán los estilos. Es como decir: "Quiero estilizar esto".

Ejemplos comunes:

  • p (para todos los párrafos)
  • h1 (para todos los títulos de nivel 1)
  • .clase (para elementos con una clase específica, súper flexible)
  • #id (para un único elemento con un ID, muy específico)

2. Propiedad

Define qué aspecto se modificará. Piensa en ello como: "¿Qué característica de esto quiero cambiar?"

Ejemplos populares:

  • color (el color del texto)
  • font-size (el tamaño del texto)
  • background-color (el color de fondo)
  • margin, padding (espaciado alrededor y dentro del elemento)

3. Valor

Especifica cómo se aplicará el cambio de la propiedad. Es la respuesta a: "¿Cómo quiero cambiar esa característica?"

Ejemplos concretos:

  • blue (para el color)
  • 16px (para el tamaño de fuente o espaciado)
  • lightgreen (para el color de fondo)
  • bold (para el grosor de la fuente)

Tips para un CSS más eficiente

  • Usa ; después de cada propiedad para evitar errores.
  • Agrupa múltiples reglas en un solo bloque para mayor orden.
  • Organiza tu CSS por secciones (ej. tipografía, layout) para mantenerlo claro y fácil de mantener.
  • En Visual Studio Code, para generar rápidamente la estructura básica de un documento HTML, escribe ! y presiona Tab.
  • Instala la extensión **Beautify** para que el código se autoformatee y mantenga la indentación clara (¡adiós al código desordenado!).
  • Instala **Live Server** para ver los cambios en tiempo real al guardar (Ctrl + S). Basta con hacer clic derecho en el archivo HTML y seleccionar "Open with Live Server".
  • Instala **VSCode Icons** para mejorar la visualización de archivos con iconos personalizados en el explorador y encontrar todo más rápido.

Ejemplo de Regla CSS

Así es como se ve una regla CSS completa:

p {
  color: blue; /* El texto de los párrafos será azul */
  font-size: 20px; /* El tamaño de la fuente de los párrafos será de 20 píxeles */
}

¡Sencillo, verdad?

Formas de agregar estilos CSS

1. Estilos en Línea (Inline Styles)

Esta forma es la menos recomendada para proyectos grandes, ya que mezcla el estilo directamente en el HTML. Se usa la propiedad style dentro de la etiqueta HTML.

Ejemplo:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Introducción CSS</title>
</head>
<body>
  <h2 style="color: red; border: 1px solid black;">Hola Mundo</h2>
  <h2 style="color: green; border: 5px solid gray;">Hola Mundo 2</h2>
</body>
</html>

2. Estilos Internos (Internal/Embedded Styles)

Aquí, los estilos se definen en la sección <head> de tu documento HTML, dentro de la etiqueta <style>. Útil para estilos específicos de una sola página.

Ejemplo:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Estilos Internos</title>
  <style>
    h1 {
      color: purple;
      text-align: center;
    }
    .destacado {
      font-weight: bold;
      background-color: yellow;
    }
  </style>
</head>
<body>
  <h1>Mi Título con Estilo</h1>
  <p class="destacado">Este es un párrafo destacado.</p>
</body>
</html>

3. Estilos Externos (External Styles)

Esta es la forma más común, limpia y recomendada. Los estilos se escriben en un archivo .css separado y se enlazan al documento HTML usando la etiqueta <link> en la sección <head>. Ideal para mantener tu código organizado y reutilizable.

Ejemplo de index.html:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Estilos Externos</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h1>Página con Estilos Externos</h1>
  <p>Este texto se estiliza desde un archivo CSS.</p>
</body>
</html>

Ejemplo de styles.css (en el mismo directorio):

body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
  color: #333;
}

h1 {
  color: darkblue;
  text-decoration: underline;
}

p {
  line-height: 1.6;
}
